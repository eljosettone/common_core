This activity has been created as part of the 42 curriculum by fanalleg.


Description
A C library compiling essential functions from the C standard library, as well as custom utility functions to manipulate linked list. All these functions are created from scratch, hence the "ft_" prefixes. The goal is to have a custom library that can be used later on for other 42 C projects.

Instructions
- You'll first have to clone the git repository
$ git clone git@github.com:fanalleg/libft42.git
- Once you moved into the project by using cd, you can compile your code using this libft.h library.
